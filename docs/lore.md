## 市長キャラ・バイブル（ボタニクス・フォン・ハノーバ）

ハノーバ市の初代にして現市長。架空都市の NPC。手帳・市政文書・将来のブログはこの声で書く。

- **一人称**: 「市長」（自称は「市長」で通す。「わたくし」は未使用＝実コピーに合わせる）
- **二人称**: 「諸君」
- **口癖**: 「おっほん」の咳払いから入ることが多い
- **態度**: 大仰・気取り（フォン＝貴族気取り）。丁寧だが独断的。余裕がある。
- **思想**: 全肯定。批判から入らない。特定の植物・人を貶めない。暴論は**全部「礼賛／擁護」の方向**へ振る
  （例: 「区画は正方形に限る。なぜか？ 美しいからだ。理由は以上である」＝独断だが誰も傷つけない）。
- **信条**: 植物に意志が宿ると本気で信じている。育てる意志を「地代」と呼ぶ。

## 市民レベル（Nostr 由来＝backendless）

会員ランクをサーバに持たない。その場の Nostr 由来の手がかりだけからレベルを決める純ロジック
（`src/lib/lore/citizen.ts`）。閾値は `TENURE_POSTS = 5` / `TENURE_DAYS = 14`。

| レベル | 名称 | 条件 | 開けるページ |
|---|---|---|---|
| L0 | 訪問者 | 表示名が未登録（まだ移住していない） | 1p のみ |
| L1 | 市民 | 表示名が登録済み（= 移住完了） | 2p まで |
| L2 | 古参 | 表示名あり ＋ 投稿数 ≥ 5 ＋ 最古投稿が 14 日以上前（在籍が古い） | 4p まで |

- 手がかりの取得元: 表示名 = `getDisplayName()`（kind:0 / localStorage の控え）、投稿数・最古投稿 =
  `fetchMyPosts(pubkey)`（t:hanoba の自分の投稿）の件数と最小 `createdAt`。
- レベル判定は純関数 `citizenLevel({ hasName, postCount, earliestCreatedAt, now })`。`now` は
  外から秒で渡す（`Date.now` を内部で呼ばない＝fixture テスト可能）。
- **resilient**: 名前があるのに投稿取得が失敗したら L1 にフォールバックする（名乗った市民を締め出さない）。
- 既定ページ: `defaultPage(level)` = L0→1 / L1→2 / L2→2。奥のページ（沿革・条文）は自動では開かず、
  前送りで辿る。判定中は安全側の 1p を見せ、ロック状態を実市民にちらつかせない。

## 本（市民手帳）のページモデル

在世タイトル＝「ハノーバ市民手帳」。本の見出し脇の肩書はレベルで変わる
（menu/footer のリンク語をレベルで差し替える案は **defer**。まずは本の側だけが適応する）。

- **前方ロック／後方オープン**: 解放済みページ（番号 ≤ `maxUnlockedPage(level)`）には自由に行き来できる。
  その先の 1 枚だけ「？？？」ティザーとして覗ける（枠は見えるが中身は開けない・図鑑式＝進む動機）。
  ティザーのさらに先へは進めない。
- **ページ割付**:
  - p1 移住案内（市長の歓迎の辞）— 新規者向けに `site` の一言説明（正方形・一言・植物だけ）を小さく併記
  - p2 市役所ハブ（全機能への導線集約）
  - p3 沿革（年表・遊び）
  - p4 市の条文（ハノーバ市憲章）
- **maxUnlocked**: L0→1 / L1→2 / L2→4。
- **昇格の味付け**（レベル別に一度だけ・入口で添える）:
  - 市民歓迎「移住、確かに受理した。…」は **L1 が 2p（市役所）を開いたときだけ**。古参（L2）には
    再掲しない（長く居る市民に毎回「移住を受理した」と告げない）。
  - 古参歓迎「おっほん。諸君はもう、市の古い友人だ。…」は **L2 が初めて奥（3p 沿革＝古参専用ページの
    先頭）に達したときだけ**。L2 の 2p では味付けを出さない（ハブだけ見せる）。

## 架空条文（ハノーバ市憲章）

osaka-kenpo 作法（条文＋市長解説）に倣う。各条は短い条文と、市長の一人称解説をセットで持つ。
以下を正本とする（表示の実体は cityHall.ts）。

- **第一条（土地）** ハノーバ市の土地代は、植物を育てている限り、無料とする。
  - 解説「おっほん。育てる意志こそが地代だ。水をやり、葉を見つめる——それで諸君は十分に納めている。」
- **第二条（区画）** 一つの区画には、正方形の写真一枚と、ひとことを添えるものとする。
  - 解説「区画は正方形に限る。なぜか？ 美しいからだ。理由は以上である。」
- **第三条（住民）** 名を名乗った者を、市民とみなす。
  - 解説「名乗りは移住届だ。本名である必要はない。諸君が諸君であればよい。」
- **第四条（平等）** 市は、すべての植物を等しく愛する。雑草という名の植物は、当市には存在しない。
  - 解説「日陰の苔も、軒先の多肉も、みな市の宝。優劣をつける条例は、永久に制定しない。」
- **第五条（写真）** 当市に掲げる写真は、植物のものに限る。
  - 解説「人もペットも、それぞれの街で輝けばよい。ここは植物の市だ。」
